Android Linux Laboratory

This lab is focused on turning an Android phone into a Linux learning environment using Termux, Ubuntu, and VNC. The goal is to practice Linux commands, package management, remote access, and system administration basics.

Tools Used

Android Termux Ubuntu VNC Linux CLI

What I Practiced

  • Installed and configured Termux
  • Set up Ubuntu inside Android
  • Practiced Linux command-line basics
  • Worked with packages, files, and directories
  • Used VNC for graphical Linux access

Goal

The goal of this lab is to build a portable Linux practice environment that supports my learning in IT support, Linux administration, networking, and automation.
